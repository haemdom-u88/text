# 前端手动验收清单

> 适用于本地启动后访问 http://127.0.0.1:5000

## 基础可用性
- 页面打开后不报错，主界面完整展示
- 顶部状态指示为“系统就绪”
- 切换“单文档/多文档/问答”面板正常

## 单文档流程
- 点击“加载示例”填充文本
- 点击“构建知识图谱”后弹出加载框
- 完成后：实体/关系计数更新、图谱可视化出现、JSON 面板有输出
- 点击“重置视图/导出图谱/复制 JSON/导出 JSON”均有效

## 多文档流程
- 选择 2 个以上文件后显示文件列表与数量
- 点击“入队并自动轮询”后显示进度条与队列状态
- 处理完成后：表格显示每文件结果、图谱更新、合并 JSON 可导出
- “仅入队”“查询队列”“下载队列结果”按钮可用

## Neo4j 相关
- 配置了 Neo4j 后，“从 Neo4j 读取”可返回节点/边
- “保存图谱”返回成功提示

## 智能问答
- 输入问题后点击“提问”，返回答案与推理链路

## 导出报告
- “导出报告”提示输入格式（json/pdf/word）
- PDF/Word 下载文件可打开

## 异常与提示
- 空文本/未选文件时有友好提示
- 请求失败时 Toast 或状态条显示错误信息
